# Medical Application

# Directory Structure
- **resources.** Original and compiled patterns of the medical example, along with auxiliary domain knowledge.
- **dataset.** This folder contains a dataset for the medical example in RTEC format, and an auxiliary definition for the maximum distance between consecutive "x" events.
- **results.** Directory of the execution logs.

# Execution Examples on the Medical Application
- ```./run_rtec.sh --app=medical```: runs RTEC with the parameters specified in "execution\ scripts/defaults.toml".

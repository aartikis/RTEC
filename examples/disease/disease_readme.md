# Disease Application

# Directory Structure
- **resources.** Original and compiled patterns of the disease example, along with auxiliary domain knowledge.
- **dataset.** This folder contains a dataset for the disease example in RTEC format.
- **results.** Directory of the execution logs.

# Execution Examples on the Medical Application
- ```./run_rtec.sh --app=disease```: runs RTEC with the parameters specified in "execution\ scripts/defaults.toml".
